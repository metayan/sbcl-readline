sbcl-readline

Exported symbols

Variable: *HISTORY-FILE*

File to keep history in. Default value is ~/.sbcl-history.

Variable: *HISTORY-SIZE*

Number of commands to keep in history. Default value is 200.

Variable: *PS1*

Either a string or a function that returns a string to be used when prompting user for the next command. The function have to take no arguments. Default is a function, returning strings "shortest-package-name[command-number]: ".

Variable: *PS2*

Either a string or a function that returns a string to be used when prompting user for the rest of incomplete command. The function have to take no arguments. Default value is "> ".

Variable: *DEBUG-PS1*

Like *ps1*, but in debugger. Default value is a function simulating default SBCL debugger behaviour.

Variable: *DEBUG-PS2*

Like *ps2*, but in debugger. Default value is "* "

Function: STIFLE-HISTORY

Syntax:

stifle-history max => nil

Arguments and values:

max---a number

Description:

Stifle the history list, remembering only the last max entries

Function: UNSTIFLE-HISTORY

Syntax:

unstifle-history => max

Arguments and values:

max---the maximum number of history entries as set by stifle-history. It is positive if the history was stifled, negative if it wasn't.

Description:

Stop stifling the history.

Function: READLINE

Syntax:

readline &key ps1 ps2 eof-value => form1, form2, ...

Arguments and values:

ps1---either a string or a function that returns a string to be used when prompting user for the command. Default is "* ".

ps2---either a string or a function that returns a string to be used when prompting user for the rest of incomplete command. Default is "> ".

eof-value---a value to return in case of the end of *standard-input* stream. By default an end-of-file condition is raised.

Description:

Prompts user for a command and returns the result as a Lisp form. Provides all readline facilities such as autocompletion and history. The result is saved in readline history. Note that user can enter several forms in one line; this results in multiple returning values.
